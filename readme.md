## Game

1 - Välj motståndare  
2 - Tänk på ett tal mellan 1 och 1000
3 - Låt datorn gissa

- Datorn gissar ett tal mellan 1 och 1000.
- Spelaren svarar med:
  - `<` om datorn gissar för lågt.
  - `>` om datorn gissar för högt.
  - `=` om datorn gissar rätt.
